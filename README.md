I use on mint & macOS.


## Dependency
* fish
* pyenv
    * python 3.10
* vscodium

### macOS
* Homebrew
* Java 21
* rbenv


## How to setup
```sh
git clone https://gitlab.com/f-matano44/dotfiles.git
cd ./dotfiles/ && sh ./setup.sh
```
