I use this on mint & Ubuntu & macOS.

## My environment

* Shell: fish
* Editor: vscodium
* Java : version 21
* Python manager: pyenv
* Ruby manager: rbenv


## How to setup

```sh
git clone https://gitlab.com/f-matano44/dotfiles.git
cd ./dotfiles/ && bash ./setup.sh
```
